# Bico website

## Installation
* Clone and `yarn install`

## Development

To start the development server just run  `yarn start`

## Build
```
yarn build
yarn build:pwa
```

## Other documentations
* [Jekyll-webpack-boilerplate](https://github.com/sandoche/Jekyll-webpack-boilerplate)
